# JumpController
Videogames motion controller for tracking jumping and ducking
![tumbnail](https://github.com/user-attachments/assets/f68a0363-9642-4dfb-a463-718d6ab489f3)

A motion control device that tracks jumps and ducks and allows to perform those actions in games using your body. ESP32 + MPU6050, the algorithm is based on classical timeseries analysis, no machine learning.
Demo: https://www.youtube.com/watch?v=s328dduBbT0
